Machine Translation at TransPerfect

In the summer of 2023, I worked for TransPerfect as an NLP intern, where I worked on an artificial intelligence research team focused on building solutions for machine translation.

Some of my responsibilities included the following:

  • Conducted exploratory data analysis on translation memory to uncover errors in pre-processing phase.
  • Identified and cleaned contamination across training and test sets to improve model evaluation.
  • Wrote scripts to measure text similarity for translation evaluation using Levenshtein distance, BLEU scores, chrF scores, COMET, TF-IDF scores, N-gram (perplexity) scores, and Cosine similarity scores.
  • Designed a test set to better represent a client domain using clustering algorithms to organize data. Used bag of words, TF-IDF and sBERT vectorization and an N-gram language model.
  • Used client data to perform domain adaptation on a Canadian French to English transformer model and the Marian toolkit to finetune the transformer model for hyperparameter optimization.
